# 面试讲点速查表

## 通用三连击（每场必讲，按公司调整侧重）
1. **上交所网络性能调优**：ARM 平台，为股票交易小包高并发低延时 UDP 调优。网络栈参数调优 + 软中断绑核 + 中断聚合 → **15万并发以下平均延时 <100us**。独立写网卡中断信息展示内核模块(~300行)。搭建 gazelle/dpdk 环境。
2. **4.19 回合高版本 bpf/btf**：理解 bpf/btf 机制，回合特性+单测，解决校验出错致近百用例失败，**合入~1000补丁**，产出 btf 机制分享文档。
3. **随机单bit翻转宕机排查**：客户小批量新机频繁宕机，调用栈无规律。分析调用栈+coredump，发现宕机规律是**内存随机单bit翻转→异常地址→多次do_page_fault→宕机**。→ 航天版引到"空间单粒子翻转(SEU)意识"。

## 银河航天⭐（星载软件工程师载荷）
- Zynq 平台 Petalinux 全栈移植（uboot/kernel/rootfs）对照星载载荷计算单元标准画像
- SPI/LCD/触摸屏/PL UartLite 设备树与驱动移植 → 星载外设驱动适配
- **AD9361 射频驱动移植** → 直接对口星载通信射频链路（差异化亮点）
- FPGA 控制代码重构 ~7000 行 → PS-PL 协同/AXI 交互能力
- kpatch 热补丁产品化 → 对应星载在轨软件上注/热补丁
- ARM RAS 调研(SDEI中断) → 航天可靠性
- 补强速学：抗辐照(SEU/SEL)、ECSS标准、PREEMPT_RT/Xenomai硬实时
- 反问：星载软件团队规模？在轨上注流程？发射节点加班天数？
