# find-job — 北京海淀前沿公司求职调研

目标：为 **王毅博**（Linux 内核/系统/嵌入式 7 年，现居北京北医三院附近）筛选 **前沿方向**（机器人 / 大模型 / 智能驾驶 / 类星链-卫星互联网）公司，硬性要求：

1. **距离近**：距 北医三院 10km 以内，优先 中关村。
2. **不忙**：排除 996 / 大小周 / 7\*24h on-call。
3. **工资越高越好**（在满足 1、2 之后）。

## 目录结构
- `resume.md` —— 原始简历存档
- `resume-航天版.md` / `resume-机器人版.md` / `resume-大模型Infra版.md` / `resume-01ai平台版.md` —— 4份针对性简历变体
- `companies.md` —— 公司清单与调研结果（最终汇总表）
- `applications.md` —— 投递进展跟踪表
- `interview-cheatsheet.md` —— 面试讲点速查表
- `interview_qa_rehearsal.md` —— 面试技术问答预演集（6家高频追问+答题要点）
- `outreach-messages.md` —— 6家内推私信+Boss招呼语
- `*-assessment.md` / `*-report.md` —— 各公司岗位深度评估报告
- `memory.md` —— 循环状态记录（避免重复抓取）

## 循环说明
每 20 分钟跑一次，每次先读 `memory.md` 决定本次抓什么，结果写入 `companies.md`，commit+push 到 `git@github.com:lucienwyb/find-job.git`。
