我们在解决什么问题

今天的 Coding Agent 还远没有到最终形态。它们会写代码，但也会迷路、重复、误解上下文、错误调用工具、无法恢复失败，在长任务中丢失目标。

我们关心的问题是：如何让 Agent 在真实软件工程任务中更稳定、更可靠、更可验证，并且能够通过 trace、评测和数据持续进化。

  • 参与 Kimi Code 核心功能和工程架构建设，构建 Coding Agent 系统；
